# BanglaTalk Client

## Quick Guide to Run BanglaTalk Client
* ``python -m venv client_env``
* ``Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass``
* ``client_env\Scripts\activate``
* ``pip install -r requirements.txt``
* Rename config_test.py to config.py and modify values of variable.
* ``python main.py``
* ``deactivate``
